3 Running the App

Run both microservices:

books $ ./gradlew run

<=========----> 75% EXECUTING [10s]
gateway $ ./gradlew run

> Task :complete/gateway:run
18:28:35.723 [main] INFO  io.micronaut.runtime.Micronaut - Startup completed in 707ms. Server Running: http://localhost:8080
